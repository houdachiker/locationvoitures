**Location de Voitures**

Le projet Location de Voitures est une application conçue pour faciliter la gestion et la location de véhicules pour une agence. Il permet aux clients de :

Consulter les voitures disponibles à la location.

Réserver un véhicule selon leurs besoins.

Suivre leurs réservations facilement.

Pour l’agence, l’application offre la possibilité de :

Gérer le parc automobile (ajout, modification, suppression des véhicules).

Suivre et organiser les réservations des clients.

Optimiser la gestion des véhicules pour éviter les conflits de réservation.

L’objectif du projet est de simplifier le processus de location de voitures, d’améliorer l’expérience utilisateur et d’assurer une gestion efficace et fluide pour l’agence.
